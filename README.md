# Sujatha’s Kitchen

A simple and elegant website for **Sujatha’s Kitchen**, showcasing homemade, preservative-free masala products prepared using traditional family recipes.
---
## About the Project
Sujatha’s Kitchen is a small homemade brand focused on delivering authentic Indian masalas with the taste of home.  
This website presents the products, customer testimonials, and a contact form for enquiries.

---
## Features
- Responsive homepage design
- Product showcase with pricing and details
- Customer testimonials carousel
- Contact form with frontend validation
- Clean and minimal UI using Bootstrap
  
---
##  Technologies Used
- HTML5  
- CSS3  
- Bootstrap 5  
- JavaScript  
- jQuery
